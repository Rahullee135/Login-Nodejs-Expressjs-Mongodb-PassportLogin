## Simple-Login-Nodejs-Expressjs-PassportLogin


## Installation
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```




## Project Required dependencies ---->

```
 $  npm i  bcrypt
 $  npm i connect-flash
 $  npm i ejs
 $  npm i express
 $  npm i express-ejs-layouts
 $  npm i express-session
 $  npm i mongoose
 $  npm i passport
 $  npm i passport-local
```
